プライバシーポリシー

本サービス「大学キャンパスの広さ比較マップ」（以下、「当サービス」）における個人情報の取り扱いについて、以下のとおり定めます。

1. 運営者

当サービスは個人が運営するウェブサービスです。お問い合わせは GitHub Issues よりお願いします。

2. 収集する情報

当サービス自体は、ユーザーの個人情報を直接収集・保存しません。ただし、以下の第三者サービスを利用しており、それらが情報を収集する場合があります。

3. Google AdSense について

当サービスでは、Google LLC が提供する広告配信サービス「Google AdSense」を利用しています。Google AdSense は、ユーザーの興味・関心に基づいた広告（パーソナライズド広告）を表示するために Cookie を使用します。

  • Google がこれらの Cookie を使用することにより、当サービスおよびその他のサイトへのアクセス情報に基づいて広告が配信されます。
  • パーソナライズド広告を無効にするには、Google 広告設定 をご利用ください。
  • Google のプライバシーポリシーは こちら からご確認いただけます。

4. OpenStreetMap・Nominatim について

地図の表示には OpenStreetMap（© OpenStreetMap contributors）のタイルデータを使用しています。場所検索機能には OpenStreetMap の Nominatim API を使用しています。これらのサービスのプライバシーポリシーは各サービスのウェブサイトをご参照ください。

5. Cookie について

当サービス自体は Cookie を設定しません。ただし、Google AdSense が広告配信のために Cookie を使用します。ブラウザの設定から Cookie を無効にすることができますが、その場合一部の機能が正常に動作しないことがあります。

6. アクセス解析

現在、当サービスは独自のアクセス解析ツールを使用していません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて改定することがあります。重要な変更がある場合は当サービス上でお知らせします。
